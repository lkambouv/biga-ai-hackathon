# Recommended VS Code Extensions

Install these extensions to get the most out of the hackathon. The setup script installs them automatically, but you can also install them manually from the Extensions panel (Cmd/Ctrl+Shift+X).

## Required

| Extension | ID | Purpose |
|---|---|---|
| **GitHub Copilot** | `github.copilot` | AI code suggestions and completions |
| **GitHub Copilot Chat** | `github.copilot-chat` | AI chat, Agent mode, inline chat |

## Recommended

| Extension | ID | Purpose |
|---|---|---|
| **Live Server** | `ritwickdey.liveserver` | Auto-reloading local server for HTML files — right-click an HTML file → "Open with Live Server" |
| **Python** | `ms-python.python` | Python language support, debugging, virtual environments |

## Optional

| Extension | ID | Purpose |
|---|---|---|
| **SQLite Viewer** | `qwtel.sqlite-viewer` | View SQLite database files directly in VS Code |
| **Markdown Preview Enhanced** | `shd101wyy.markdown-preview-enhanced` | Better markdown preview with diagrams and math |
| **REST Client** | `humao.rest-client` | Send HTTP requests directly from VS Code (useful if building an API) |

## How to Install Manually

1. Open VS Code
2. Press **Cmd+Shift+X** (macOS) or **Ctrl+Shift+X** (Windows)
3. Search for the extension name
4. Click **Install**

Or from the terminal:
```bash
code --install-extension github.copilot
code --install-extension github.copilot-chat
code --install-extension ritwickdey.liveserver
code --install-extension ms-python.python
```
