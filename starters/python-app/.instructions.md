- This is a Python web application using Flask with Jinja2 templates
- No external AI/LLM API calls — all logic is in Python
- Data is stored in in-memory Python lists and dicts (resets on server restart)
- Keep code simple with clear comments explaining each section
- Use descriptive function and variable names
- Flask auto-escapes template variables ({{ }}) to prevent XSS
- Templates are in the `templates/` folder
- Run with `python app.py` (debug mode on port 5000)
