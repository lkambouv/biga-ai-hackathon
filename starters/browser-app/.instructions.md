- This is a browser-only application using vanilla JavaScript, HTML, and CSS
- No frameworks, build tools, or external dependencies
- Data is stored in localStorage (persists across page refreshes)
- Use `saveItems()` after modifying the `items` array to persist changes
- No external API calls — all logic runs in the browser
- Keep code simple with clear comments explaining each section
- Use descriptive function and variable names
- Use `escapeHtml()` when rendering user input to prevent XSS
